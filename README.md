# OverTheWire: Bandit Wargame Solutions

This repository contains my personal writeups, scripts, and command logs for the [OverTheWire: Bandit](https://overthewire.org/wargames/bandit/) wargame.  

As a software developer, I use challenges like this to keep my Linux CLI, system administration, and basic security fundamentals sharp. This repository serves as a reference guide for the tools and concepts encountered throughout the 34 levels.

## Disclaimer

In accordance with the OverTheWire community guidelines, **this repository does not contain the actual level passwords.** The purpose of this repo is to document the *methods*, *commands*, and *scripts* used to solve each level. If you are currently playing the wargame, I highly encourage you to try solving the levels yourself before looking at these methods!

## Tools & Prerequisites

To execute the commands found in this repository, you will need:

* **SSH client**: To connect to the wargame servers.
* **Git**: For the version control challenges in the later levels.
