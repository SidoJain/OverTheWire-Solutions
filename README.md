# OverTheWire Wargames

This repository tracks my progress, notes, and solutions for the [OverTheWire](https://overthewire.org/wargames/) security wargames. These games are designed to help learn and practice security concepts, starting from basic Linux command-line skills and moving up to advanced exploitation.  

## Repository Structor

Each wargame is organized into its own subdirectory. \
Currently, this repository includes:

* **[Bandit](https://overthewire.org/wargames/bandit/)**: The beginner friendly wargame aimed at teaching the basics of the Linux command line, permissions, and basic scripting.

## Disclaimer & Spoiler Warning

This repository contains complete solutions, scripts, and level passwords. If you are currently playing the OverTheWire wargames, please turn back now! The entire point of the games is to learn by struggling through the challenges yourself. Reading the solutions here will ruin the fun and the learning process.
